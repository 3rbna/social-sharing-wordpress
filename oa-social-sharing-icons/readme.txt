== Installation ==

= Plugin Installation =
1. Upload the plugin folder to the "/wp-content/plugins/" directory of your WordPress site,
2. Activate the plugin through the 'Plugins' menu in WordPress,
3. Visit the "Settings\Social Sharing Icons" administration page to setup the plugin. 

= API Connection =
The social network APIs are constantly changing and being updated. We monitor these changes and automatically 
update our APIs, so that you can be sure that Social Sharing Icons will always run smoothly and with the most up-to-date 
API calls. 

In order to enable the plugin you must connect with the OneAll API and create a free account at https://app.oneall.com

== Frequently Asked Questions ==

= Do I have to add template tags to my theme? =

You should not have to change your templates. 
The Social Sharing Icons seamlessly integrate into your blog by using predefined hooks.

= Does the Social Sharing Icons can be embedded through a shortcode? =

The Social Sharing shortcode `[oa_social_sharing_icons]` can be used in any page or post within your blog.
The shortcode will automatically be replaced by the icons of the chosen social network providers.

= I have a custom template and the plugin is not displayed correctly =

The plugin uses predefined hooks. If your theme does not support these hooks,
you can add the Social Sharing Icons manually to your theme by inserting the following code 
in your template (at the location where it should be displayed, i.e. after the posts).

`<?php do_action('oa_social_sharing_icons'); ?>`

Do not hesitate to contact us if you need further assistance. 

= Do I have to change my Rewrite Settings? =

The plugins does not rely on mod_rewrite and does not need any additional rules.
It should work out of the box.
